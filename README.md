🚀 Proje Hakkında / About

WMC, modern web teknolojileri ve Mineflayer kütüphanesini bir araya getiren bir Network Stress Test simülatörüdür[cite: 5]. Dağıtık sistemlerin Minecraft protokolü üzerindeki davranışlarını analiz etmek ve sunucu dayanıklılığını ölçmek amacıyla geliştirilmiştir[cite: 5].
✨ Teknik Özellikler / Technical Features

    Sync Control: Tüm birimlere milisaniyelik hassasiyetle toplu hareket ve etkileşim komutları gönderimi[cite: 5].

    Real-time Dashboard: WebSocket mimarisi üzerinden anlık bot durum takibi ve canlı sistem logları[cite: 5].

    Dual Language Support: Tek tıkla TR/EN arayüz geçişi[cite: 5].

    Advanced Configuration: Gecikme süreleri (ms), bot limitleri ve protokol versiyonları üzerinde tam kontrol[cite: 5].

    Safety Kill-Switch: Tek tuşla tüm ağ bağlantılarını güvenli bir şekilde sonlandırma protokolü[cite: 5].

🛠️ Kurulum / Installation
📋 Ön Koşullar

    Node.js (LTS Sürümü)

    NPM (Node Package Manager)

🔧 Çalıştırma Adımları

    Depoyu klonlayın: git clone https://github.com/ubeytkzlts/WMC-Bot.git

    Proje dizinine girin: cd WMC-Bot

    Bağımlılıkları yükleyin: npm install express socket.io mineflayer

    Paneli başlatın: node server.js

Kurulum tamamlandıktan sonra tarayıcınızdan http://localhost:3000 adresine giderek panel arayüzüne erişebilirsiniz.
